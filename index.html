Welcome to master.po-util.com!

This sub-domain of po-util.com is a mirror of the master branch of the po-util project.

View the project on GitHub.

The purpose of this mirror is to provide a convenient URL for the po-util installation script. With this mirror, po-util can be installed on Linux and macOS with the following:

$ bash <(curl -sL https://master.po-util.com/install)

For online documentation please refer to docs.po-util.com.
Additional support is available at gitter.im.

Thank you for using po-util.
